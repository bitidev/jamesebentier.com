# AGENTS.md

Instructions for AI coding agents (OpenAI Codex, GitHub Copilot, Cursor,
Windsurf, Amp, Devin) working in this repository. See <https://agentsmd.io>.

Token-optimized CLI output is handled globally by **rtk** (hooks + agent
instructions). Prefer `rtk <command>` for verbose shell work when a hook is
not rewriting the call automatically.
